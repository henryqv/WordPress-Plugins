SEO Personalizado es un plugin de WordPress que permite personalizar el SEO de tus páginas y publicaciones.

== Descripción ==

SEO Personalizado es un plugin de WordPress que permite personalizar el SEO de tus páginas y publicaciones. Con este plugin, puedes:

* Cambiar el título de las páginas.
* Agregar una meta descripción.
* Elegir si deseas agregar los meta noindex y nofollow.
* Agregar schemas personalizados.
* Definir una URL canonical.

== Instalación ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/seo-personalizado`, o instala el plugin directamente desde el repositorio de plugins de WordPress.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Configura las opciones de SEO en cada página o publicación.

== Preguntas Frecuentes ==

= ¿Cómo cambio el título de una página? =

Ve a la página o publicación que deseas editar, y encontrarás un campo para el título en la sección de SEO Personalizado.

= ¿Puedo agregar una meta descripción? =

Sí, puedes agregar una meta descripción en el campo correspondiente en la sección de SEO Personalizado.

= ¿Qué son los meta noindex y nofollow? =

El meta noindex le dice a los motores de búsqueda que no indexen la página, mientras que el meta nofollow le dice que no sigan los enlaces en la página.

= ¿Cómo agrego schemas personalizados? =

Puedes agregar schemas personalizados en el campo de texto proporcionado en la sección de SEO Personalizado.
